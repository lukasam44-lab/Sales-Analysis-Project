# Sales Analysis Project

## Overview
Analysis of retail sales data across multiple store locations 
using Python.

## What I analyzed
- Sales trends by store location
- Hourly sales patterns
- Product category and type performance
- Daily and weekly sales patterns

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Files
- `coffee shop sales.ipynb` — main Jupyter notebook
- `coffee shop sales.csv` — dataset used
